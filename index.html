HW16-class-inheritance
1. Создайте на вашем github репозиторий с именем HW16-class-inheritance. Все результаты нужно запушить в ваш репозиторий и прикрепить ссылку на hillel портале.
2. Создайте index.html в котором подключите js script.
3. Необходимо реализовать родительские часы и дочерние часы с помощью классов.
3.1 Реализовать класс Clock.
3.2 Реализовать минимум два дочерних инстанса. Нужно создать классы которые будут наследовать базовые методы и свойства от Clock.
3.3 Реализовать полиморфизм для метода отображения. Каждый чайлд который наследуется от родительского Clock должен предоставлять свой формат отображения.
3.4 При клике по часам нужно отображать разный формат времени. Для отображении разного формата времени нужно использовать те часы которые соответствуют нужному формату.
